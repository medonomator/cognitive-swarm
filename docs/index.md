features:
  - title: Signal Bus
    details: Typed signals (discovery, proposal, vote, challenge, doubt) flow through a shared bus. Agents react, not follow scripts.
  - title: Consensus Engine
    details: 5 strategies - confidence-weighted, Bayesian, entropy-based, hierarchical, and voting. Dissent is preserved, not discarded.
  - title: 19 Math Modules
    details: Shannon entropy, free energy, Bayesian inference, causal inference, game theory, optimal transport, and 13 more. Math decides when to stop.
  - title: Resilience
    details: Exponential backoff retry, circuit breaker, token budget, and resumable checkpoints. Production-ready out of the box.
  - title: OpenTelemetry
    details: 20 span types covering every event - signals, votes, consensus, math analysis, topology changes. Full Jaeger/Zipkin support.
  - title: A2A + MCP Protocols
    details: Expose any swarm via HTTP as a standard A2A agent. Give agents external tools via MCP servers.
  - title: Thompson Sampling
    details: Agents adaptively select strategies using Thompson Sampling bandit. The swarm learns which approaches work per context.
  - title: Self-Evolving
    details: Agents detect expertise gaps and vote to spawn new specialists. The swarm grows itself.
